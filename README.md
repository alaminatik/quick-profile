## Install Application

This application is make with php framework laravel with incredible MySql database

```bash
    git clone https://github.com/alaminatik/quick-profile.git
    cd quick-profile
    composer install
    mv .env.example .env
```
### Than Run This Command
```php
    php artisan key:generate
    php artisan serve
```
Than visit 
```code
    http://127.0.0.1:8000/
```
